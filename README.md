## Setup Instructions

1. Clone the repository
2. Install dependencies: `npm install`
3. Development mode: `npm run dev`
4. Build: `npm run build`
5. Start: `npm start`

## Technology Stack

- Material-UI
- Vite

- electron-store (Data Persistence)
